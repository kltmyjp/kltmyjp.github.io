動画チャンネルのご紹介

以下は筆者のYouTubeチャンネルとなります。

  • KL Taro Channel - YouTube（外部サイト）

上記にて直近にアップロードした動画を公開しております（※本数は時期によって異なります）。

註

古いタイトルは動画一覧から削除していますが、一覧から消えても動画自体はYouTubeのサーバに現存しています。

公開済み動画の限定公開への切り換えは筆者の意図によるものであるため、ここでもそのリスト（URL）は公開しません。

しかしながら、下記の方法を実行することで動画一覧のデータ取得が可能です。→実際に試してみましたが、一般公開の情報しか取得できませんでした*1。

  1. YouTube Data API v3 の有効化
  2. 認証情報の取得（APIキーやOAuth関係）
  3. 取得したデータをExcelやGoogleスプレッドシート等に保存するコードを書く
  4. 必要に応じてライブラリをインストール
  5. 作成したスクリプトまたはマクロを実行

上記にて動画タイトル、URL、アップロード日、概要などのデータが取得可能です。もちろん、取得したURLから各動画にアクセスすれば自由に視聴可能となります。ご興味がございましたらどうぞお試し下さいませ。

期間限定公開の理由

元々筆者の動画は、広く沢山の人に観てもらいたい、という意図で作っているのではなく、「誰に観てもらわなくても構わないので、単にその時に言いたいことを好き放題に言いたい」という動機に過ぎません。

そしてそれをわざわざ公開しているのは、「言いたいことを外に向かって言った」という満足感を得たいからで、要はちょっとした自己顕示欲の一種だと思われます。

ですが、他者にとって役に立つ性質のものではないことを自認している以上、長い間それを公に放置しておくのも恥曝しのような気がして後ろめたいのです。

せめて一定期間経過後に動画を公の場から退けることで、人様からの赦しを得たい――そのような心持ちから上記のような措置を取らせていただいております。

何卒悪しからずご了承下さい。
